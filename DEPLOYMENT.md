# Deployment Guide - Linfy Cyber Security

## Quick Deploy to Vercel (Recommended)

### Step 1: Push to GitHub

\`\`\`bash
git init
git add .
git commit -m "Initial commit: Linfy Cyber portfolio"
git branch -M main
git remote add origin https://github.com/linfordlee14/linfy-cyber-security.git
git push -u origin main
\`\`\`

### Step 2: Deploy on Vercel

1. Go to [vercel.com](https://vercel.com)
2. Sign in with GitHub
3. Click "New Project"
4. Select `linfy-cyber-security` repository
5. Vercel auto-detects Next.js
6. Click "Deploy"

**Your site is live at:** `https://linfy-cyber-security.vercel.app`

### Step 3: Add Custom Domain (Optional)

1. Vercel Dashboard → Settings → Domains
2. Add your domain
3. Update DNS records
4. SSL auto-configured

## Alternative: Deploy to Netlify

1. Push to GitHub (same as above)
2. Go to [netlify.com](https://netlify.com)
3. Click "New Site from Git"
4. Select GitHub repo
5. Build settings:
   - Command: `npm run build`
   - Publish: `out`
6. Deploy

## Troubleshooting Deployment

### Build fails on Vercel
- Check Node.js version (18+ required)
- Clear cache: Settings → Clear Build Cache → Redeploy
- Check for TypeScript errors locally: `npm run build`

### .gitignore issues
- Ensure `.gitignore` includes:
  \`\`
  node_modules/
  .next/
  .env.local
  .DS_Store
  \`\`

### Contact form not working
- Verify API key in environment variables
- Test locally first: `npm run dev`
- Check browser console for errors
- Verify CORS settings if using external service
